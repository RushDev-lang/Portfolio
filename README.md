# Personal Portfolio Website

This is my personal portfolio website, designed to showcase my resume, skills, and projects in a clean and professional layout. The site is fully responsive and features smooth navigation for an optimal user experience.

## Features
- Responsive design for all devices
- Showcase of skills, experience, and projects
- Clean, modern, and professional layout
- Interactive elements for better engagement

## Technologies
- HTML, CSS, JavaScript
- Optional: React or other frameworks (if used)

Clone the repository and open `index.html` in your browser to view the portfolio.
